# User Management App

Приложение для управления списком пользователей, построенное на Angular.

## Стек технологий
- Angular 20+
- Ng-ZORRO (UI Kit)
- TypeScript
- SCSS
- JSONPlaceholder API

## Основные функции
- Просмотр списка пользователей (с пагинацией и фильтрацией).
- Просмотр детальной информации о пользователе.
- Создание и редактирование пользователей (Reactive Forms).
- Удаление пользователей.

## Запуск
1. Установите зависимости: `npm install`
2. Запустите проект: `npm start`
3. Откройте в браузере: `http://localhost:4200`
